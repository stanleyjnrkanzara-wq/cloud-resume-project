 Cloud Resume Project

 Overview

This project demonstrates a CI/CD pipeline that automatically deploys infrastructure to AWS using Terraform and GitHub Actions.I fully did it with my phone using termux.

 Architecture

- GitHub → CI/CD pipeline
- GitHub Actions → automation
- Terraform → infrastructure provisioning
- AWS S3 → hosting

Features

- Automated deployment on push
- Infrastructure as Code
- Cloud-native setup

Tech Stack

- AWS
- Terraform
- GitHub Actions
- Linux (Termux)
